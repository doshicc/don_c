# Лабораторная работа №1: Основы языка C

## Общая информация

- **Цель работы**: Изучение основ программирования на языке C
- **Темы**: типы данных, операции ввода/вывода, арифметические операции, указатели
- **Библиотеки**: `stdio.h`, `limits.h`, `stdlib.h`
- **Компилятор**: GCC

## Инструкция по запуску

### Компиляция и запуск через терминал

```bash
clang task1.c -o task1
./task1
```

### Запуск в VS Code

1. Установите расширение "C/C++" от Microsoft
2. Откройте файл `.c`
3. Нажмите кнопку ▶ в правом верхнем углу
4. Или используйте `Fn + F5`
